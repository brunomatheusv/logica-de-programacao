## 📁 Organização do repositório

Os exercícios estão organizados por tema/conceito, seguindo a progressão do curso:

| Pasta | Conteúdo |
|---|---|
| `01-variaveis-e-operadores` | Declaração de variáveis, operações aritméticas e entrada/saída de dados |
| `02-condicionais` | Estruturas `Se/Senão` e condicionais aninhadas |
| `03-escolha-caso` | Estrutura `Escolha/Caso` para menus e múltiplas opções |
| `04-loops` | Estruturas de repetição: `Para`, `Enquanto` e `Repita/Até` |
| `05-procedimentos` | Modularização com procedimentos, passagem por valor e por referência |
| `06-funcoes` | Funções com retorno de valor |
| `07-strings-e-caracteres` | Manipulação de strings e operações com caracteres |

---

## 🛠️ Tecnologia utilizada

- **VisuAlg 3.0** — interpretador de pseudocódigo em português utilizado para o aprendizado de lógica de programação

---

## 📚 Alguns exercícios em destaque

- **Sequência de Fibonacci** — implementada de três formas: iterativa, com procedimento e com função
- **Calculadora de IMC** — com classificação por faixas usando condicionais aninhadas
- **Fatorial** — implementado com loop e com função recursiva
- **Detector de Mais Pesado** — uso de procedimentos com interface interativa
- **Analisador de Strings** — manipulação de texto com funções nativas do VisuAlg
- **Super Contador** — menu interativo com `Escolha/Caso` e loops aninhados
